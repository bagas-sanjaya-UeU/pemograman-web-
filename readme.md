## Docker dokumentasi

Docker compose

```bash
  docker-compose up -d
```

Execute Bash

```bash
  docker exec -it [NamaContainerPHP] bash
```

Create project laravel

```bash
  composer create-project laravel/laravel .
```

Hak akses storage

```bash
  chmod 777 -R storage/*
```

## Tech Stack

**Client:** Bootstrap 5

**Server:** Nginx, Laravel 9
